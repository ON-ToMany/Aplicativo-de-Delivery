# 📦🍽️ FriendlyFood

> Compartilhando refeições, espalhando solidariedade.

---

## 🚀 Sobre o Projeto

O **FriendlyFood** é uma API desenvolvida com o propósito de combater o desperdício de alimentos e promover o acesso à alimentação de forma consciente e solidária.

Mais do que um sistema, o projeto busca gerar impacto social, conectando tecnologia, nutrição e responsabilidade.

---

## 🌱 Missão

**Tecnologia que alimenta, performance que transforma.**

---

## 🎯 Objetivo

- Reduzir o desperdício de alimentos  
- Promover consumo consciente  
- Facilitar o compartilhamento de refeições  
- Gerar impacto social positivo  

---

## 🧠 Funcionalidades

- 📦 Cadastro de alimentos  
- 📊 Informações nutricionais detalhadas  
- 🥗 Classificação nutricional (Nutri-Score)  
- 👤 Cadastro de usuários  
- 🏷️ Categorias de alimentos  
- 🔐 Autenticação com JWT
